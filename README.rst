Quick start
-----------

1. Add "django-ismigrated" to your INSTALLED_APPS setting like this::

    INSTALLED_APPS = [
        ...
        'django-ismigrated',
    ]

2. Run `python manage.py ismigrated` to determine if there are migrations to run.
